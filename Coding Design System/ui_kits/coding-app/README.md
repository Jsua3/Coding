# UI Kit — Coding app

Recreacion interactiva de la app de aprendizaje (creada desde cero: las fuentes estaban vacias, ver readme raiz).

Flujo demo en `index.html`: **Login → Dashboard (materias + stats) → Curso (Bases de datos I) → Leccion (SQL + quiz con dialogo de resultado y toast de XP)**.

Archivos: `AppShell.jsx` (NavBar flotante + iconos), `LoginScreen.jsx`, `DashboardScreen.jsx`, `CourseScreen.jsx`, `LessonScreen.jsx`, `data.js` (datos de ejemplo). Todos componen los primitivos de `components/` via `window.CodingDesignSystem_2ecb3a`.
