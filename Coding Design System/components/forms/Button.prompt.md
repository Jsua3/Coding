Pill Liquid Glass button — primary (blue gradient + glow) for the main action, secondary (glass) for everything else.

```jsx
<Button variant="primary" size="md" onClick={fn}>Continuar leccion</Button>
<Button variant="secondary">Ver temario</Button>
```

- Variants: primary | secondary | ghost | danger. Sizes: sm 36px / md 44px / lg 52px. Press = scale(0.96) spring.
