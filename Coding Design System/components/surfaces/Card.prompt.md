Glass content card with optional eyebrow/title/subtitle/footer and hover lift — the default block for courses, lessons and stats.

```jsx
<Card eyebrow="Bases de datos I" title="Modelo relacional" subtitle="12 lecciones" tint="cyan" onClick={fn} />
```

- `tint` matches subject hues; `footer` takes actions; set `hoverable={false}` for static cards.
