Foundational Liquid Glass surface — use it as the container for any grouped content sitting on the aurora background.

```jsx
<GlassPanel tint="blue" strength="strong">
  <h3>Bases de datos I</h3>
</GlassPanel>
```

- `tint`: none | blue | cyan | violet (refracted accent light)
- `strength`: subtle | default | strong (fill + shadow depth)
- `specular`: top sheen, on by default
